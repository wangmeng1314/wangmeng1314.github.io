maven入门学习

发表于 2016-07-10   |   分类于 工作   |     |   阅读次数

        maven是一款强大的项目构建工具，为我们管理项目带来了极大的方便，让我们一起来学习吧！

maven基础

maven的功能

  • 构建项目（Builds）
  • 依赖管理（Dependencies）
  • 配置管理（SCMs）
  • 发布管理（Releases）
  • 文档编制（Documenation）
  • 报告（Reporting）

maven的特点

  • 微内核设计（插件机制，解析xml文档，管理生命周期，管理maven插件等，插件也是从maven仓库中获取到的）
  • 约定优于配置
指定目录 资源类型
源代码 java
资源文件 resources
测试代码 test
字节码文件 classes
jar/war包 target

在没有配置的情况下，maven所使用的核心插件也是默认的，当你遵守约定的时候，你所需要做的事情就很少了。

安装maven

  1. maven.apache.org–下载压缩包
  2. 拷贝到指定目录，解压
  3. maven目录结构的详解
目录名称 作用
bin 运行maven所需要的脚本文件，环境变量的配置也就是这个目录
boot 创建maven运行所需要的加载器的jar文件
conf settings.xml
lib maven的核心jar文件

配置环境变量

M2_HOME

测试

使用命令行构建maven项目

1.首先确立mvn的工作目录。
2.运行mvn archetype：generate

这是一个管理项目模板的插件，在运行的时候可能会下载很多的组件

3.grounp-id 项目所属组织的标示

artifactId 项目的标示，实际上对应的是项目的名称

4.pom文件定义的项目的对象模型
（pom文件的话需要我们下功夫研究）

5.最后验证我们构建的项目

首先mvn package

其次查看target目录下有无生成的jar包

java -cp grounpId.App

输出HelloWorld测试完成

总结

        上面我们看到了maven的一些介绍及其基本特点，也了解了maven的基本配置，当然，也使用命令行建立了一个典型的maven项目，进行了简单的测试。可以说，用好maven，对我们管理项目将起到非常大的帮助！
