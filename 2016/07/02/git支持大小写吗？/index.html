git支持大小写吗？

2016-07-02
#git
git学习

        git使用中遇到的坑-git支持大小写吗？

前言

        首先要问大家一句git到底支持大小写吗？有没有遇到过大小写方面的问题呢？如果没有，说明你还没踩到坑，但是如果有了，那恭喜你，看了这篇博文，我们就明白了！让我们开始吧！插播：我爱笨媳妇。

问题背景

        在刚进入公司开发接口的过程中，我将java中的包名进行了大写，这个时候，我发现，我在我本地改成小写之后推送到服务器上的时候是推送不上去的（当然，包名小写是常识），为此还被教育过，当然是被我的师姐教育了。

解决方案

        我百思不得解啊，到底是怎样的原因让我的推送无法成功提交到服务器上去呢，我的心里好着急！后来查询资料得知，git默认是不支持大小写的，我心里的疑问才算解开！但是其实，我们是可以设置其支持大小写的，方法如下：

git config core.ignorecase false

        但是，我们还有别的解决方案：比如删掉那个文件重新添加提交，可是总感觉这种办法不是什么长久之计。
        so推荐使用第一种方式。

总结

        这次的坑把我坑的不轻，虽说看上去是一个很小的问题，但是其实还是相当麻烦的，你懂得！这也让我明白了在平时的开发中一定要仔细小心！
